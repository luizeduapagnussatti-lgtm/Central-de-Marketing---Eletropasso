# Tailwind Encarte (Sprint E)

Build utilitario CSS prefixado `ep-` para templates de encarte.

```bash
npm run build:encarte-css
```

Saida: `assets/encarte/tailwind.encarte.css` (commitada, sem CDN).

## Go/no-go (Sprint E4)

**Manter Tailwind** apenas para novos slots de layout (`ep-grid`, `ep-absolute`, etc.) no modelo_03.

**Manter `encarte-utilities.css`** para efeitos de varejo (.foto-flutuante, .preco-3d, .card-glow) — nao migrar para Tailwind.

Nao expandir Tailwind para `public/css/app.css` (shell admin permanece vanilla).
